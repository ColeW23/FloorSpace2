FloorSpace

FloorSpace is a web application designed to allow tenants to register with their residence information, create new maintenance tickets, and access current and past maintenance tickets.

Admins, or landlords, are able to use this web application to access all open tickets and resolve them.

The goal of the web application is to design an easy to use interface that works with all platforms to create a simple way to tackle issues in tenant living environments.
